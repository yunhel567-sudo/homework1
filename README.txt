程序说明：
第一周
ga-1.m 为自拟适应性函数的遗传算法演示，单目标
ga-pareto.m 在ga-1的基础上改为多目标，添加了帕累托算法

calculateCollisionRisk.m 为危险判断函数，输入输出格式见函数内注释
verify_cpa.m 为危险判断函数的一个验证代码，分别展示了危险和不危险两种情况，并绘制航路图

decideEvasiveAction.m 为会遇局面判断函数，输入输出格式见函数内注释
其中避让规则如下：
避碰规则提炼为以下典型的决策局面：
1.无紧迫危险：保向保速（直行）
2.对遇局面 (Head-on)：两船面对面，航向相反。规则要求双方均向右避让（右转）
3.交叉相遇 - 本船让路 (Give-way)：目标船在本船右舷，本船为让路船，需避免横越其船艏（右转，从其船尾绕行）
4.交叉相遇/被追越 - 本船直航 (Stand-on)：目标船在左舷或后方，本船为直航船，规则要求保向保速（直行）
5.本船追越他船 (Overtaking)：本船速度快，追越前方同向船只。如果他在我们正前方偏右，为了不横向切入其航线，通常选择从其左舷超船（左转）
demo_colregs_plot.m 为会遇局面判断函数的验证代码，验证了四种会遇情况，并绘制航路图


第二周
核心模块说明
ais_sync.h / ais_sync.c	 为数据接收与时空对齐模块；接收两船的原始 AIS 数据（经纬度、航速、航向、时间戳等），利用投影消除距离失真，并将目标船的坐标插值对齐到本船的时间基准上。输出统一格式的相对运动参数（对齐后的距离、真方位、速度矢量）

collision_risk.h / collision_risk.c 碰撞危险评估模块 (CPA/TCPA)；接收时间对齐后的相对运动参数，计算两船的最近会遇距离 (DCPA) 和到达最近会遇点的时间 (TCPA)，根据设定的安全阈值，将当前状态量化为三个风险等级：0-无风险，1-小风险，2-大风险 

colregs_decision.h / colregs_decision.c 避碰规则决策模块 (COLREGs)；根据碰撞风险等级，若存在风险，则依据COLREGs，判断当前的会遇局面（对遇、交叉相遇、追越）输出本船作为让路船或直航船的初步避让动作（0-直行，1-左转，2-右转）

path_decision.h / path_decision.c 避碰路径规划与决策模块 (GA/VO)；接收前序模块的风险等级和避让方向
小风险时: 采用遗传算法 (GA) 进行全局寻优，输出兼顾经济性和安全性的四个参数：直航延迟时间、转向角、避让持续时间、复航转向角
大风险时: 采用速度障碍法 (VO) 进行紧急避让，计算并输出能够立即脱离碰撞危险锥的极限调整航向角

main.c 系统主程序；整合上述所有模块，模拟真实的导航周期。包含全局状态变量（用于判断是否已在避碰进程中），并内置了多个测试场景（无风险、小风险、大风险）以验证系统的整体运行逻辑，之后会修改成函数便于后续代码编写

CMakeLists.txt CMake 构建配置文件

show.m 数据为main初始化的船只数据和输出结果，在matlab上绘图展现两种情况避让效果图
